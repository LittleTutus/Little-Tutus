# 🩰 Little Tutus Invoice Generator

A free invoice generator for Little Tutus dance studio.

## How to deploy on Vercel (free, 5 minutes)

1. Go to [github.com](https://github.com) and create a free account
2. Create a new repository called `littletutus-invoice`
3. Upload all these files (drag & drop works!)
4. Go to [vercel.com](https://vercel.com) and sign in with GitHub
5. Click **Add New Project** → select your repository
6. Click **Deploy** — Vercel does the rest!
7. You get a free URL like `littletutus-invoice.vercel.app`

## Add to your phone home screen

1. Open your Vercel URL in Safari (iPhone) or Chrome (Android)
2. Tap **Share** → **Add to Home Screen**
3. Name it "Little Tutus" and tap Add
4. It now appears as an app on your phone! 🩰
